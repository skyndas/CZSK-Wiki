Instalace Apache na Váš lokální počítač

Pro správné zobrazení CZSK - Wiki je nutné mít nainstalovaný Apache server ve Vašem lokálním počítači. Apache server je obsahem progamů XAMPP a WAMP

Instalace XAMPP

1.

Stáhněte XAMPP z následujících webových stránek https://www.apachefriends.org/download.html .

Distribuce XAMPPu obsahuje Apache, MySQL, PHP, phpMyAdmin, OpenSSL, XAMPP Control Panel, Webalizer, Mercury Mail Transport System, FileZilla FTP Server, Tomcat, Strawberry Perl Portable

2.

Spusťtě stažený soubor

3.

Po první instalaci XAMPPu Vám instalační proces nabídne zdali chcete zobrazit "Control panel" (Kontrolní panel). Stačí kliknout pouze na tlačítko "Finish" a jdeme na vlastní spuštění Apache serveru.

4.

Po zobrazení "Control panel" (Kontrolní panel), můžeme nyní už spustit vlastní Apache server.

Server Apache spustíme kliknutí na tlačítko "Start" modulu "Apache". Ostatní moduly nepotřebujeme spouštět.

5.

Pokud chceme spouštět server Apache vždy po spuštění aplikace XAMPP, zobrazíme si "Config" v "Control Panel", kde zaškrtnutím položky "Apache" v oddíle "Autostart of modules" zvolíme automatické spouštění Apache serveru. Klikneme na tlačítko "Save" a ukončíme "Control panel".

6.

Po instalaci a spuštění Apache serveru nyní provedeme rychlou kontrolu správné instalace.
Otevřeme webový prohlížeč (Firefox, Chrome ....) a do příkazového řádku zadáme adresu http://localhost .

V případě, že se nám zobrazí logo XAMPP s výběrem jazyka byla instalace XAMPPu úspěšná.

7.

Nyní přejdeme do složky "c:\xampp\htdocs\" umístěné v lokálním počítači, kde vytvoříme novou složku "czsk-wiki".

Do složky "czsk-wiki" zkopírujeme obsah zkomprimovaného souboru CZSK - Wiki znalostní databáze.

Otevřeme webový prohlížeč (Firefox, Chrome ....) a do příkazového řádku zadáme adresu http://localhost/czsk-wiki .
